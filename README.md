## 🚀 Technical Features

* **Responsive Art Direction:** Expert use of the HTML `<picture>` element or CSS background-image switching to serve optimized images for different viewports.
* **Mobile-First Workflow:** A fluid, stacked architecture designed for small screens that expands into a balanced dual-column desktop layout.
* **Apple-Style Minimalist UI:** Meticulous attention to white space, clean typography (Montserrat & Fraunces), and subtle border radii.
* **Semantic HTML5:** Built with accessibility in mind, ensuring a clear document structure for screen readers and SEO.
* **Modern CSS Layout:** Strategic use of **Flexbox** or **Grid** to manage the card's internal content alignment and spacing.

## 🛠️ Tech Stack

- **Structure:** HTML5 (Semantic)
- **Styling:** CSS3 (Flexbox, Custom Properties)
- **Deployment:** Vercel

## 🧠 Key Learnings

Building this product card refined my ability to handle **precision layout shifts** and **web typography**.

**Main Takeaways:**
- Implementing **responsive imagery** to ensure visual high-fidelity across all device types.
- Mastering **CSS Custom Properties** to manage a professional color palette and layout spacing.
- Fine-tuning **font-weight and line-height** to match the original design's premium feel.
- Organizing a clean, professional codebase focused on maintainability and standard best practices.
